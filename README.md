# ProCalculator | Laboratory Work #1: Development of a static web application interface
## Project Overview
DevCalc is a static, multi-page web interface developed as an educational project for a university Web Design course. It serves as a dual-purpose tool, providing both standard arithmetic capabilities and a specialized mode tailored for software developers.

The project focuses on implementing a modern, responsive UI/UX using flexible grid systems and consistent styling across various device types.

## Assignment Goals
* Develop a multi-page static web application.

* Ensure full responsiveness across mobile, tablet, and desktop screens.

* Utilize modern CSS frameworks for layout and component design.

* Maintain a unified visual aesthetic throughout the interface.
